## Project Objective

This project analyzes credit risk modeling techniques by comparing traditional logistic regression with machine learning approaches in a financial risk framework.

The main objective is to estimate the Probability of Default (PD) of borrowers and evaluate model performance under risk management criteria.

---

## Methodology

The project will include:

- Exploratory Data Analysis (EDA)
- Logistic Regression (baseline model)
- Random Forest
- Gradient Boosting
- Model comparison using:
  - ROC-AUC
  - Precision-Recall
  - Confusion Matrix
  - Cross-validation

---

## Economic Perspective

Beyond predictive accuracy, this project evaluates:

- Interpretability
- Regulatory suitability
- Business impact
- Risk management implications
